My Experience!

At first, using a code editor felt confusing and difficult, but once I got used to it, it actually made things a lot easier. I used VS Code, and it really helped me organize my files, catch mistakes, and write code faster. It also provides presets which is huge help!

One thing I liked was how it highlights different parts of the code in different colors. That made it way easier to read and understand.
